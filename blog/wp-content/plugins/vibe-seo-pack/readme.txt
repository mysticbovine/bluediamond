== Description ==

WordPress SEO have never been easier! **Vibe SEO Pack** is a powerful and easy to use **WordPress SEO plugin**.

Vibe SEO Pack is not just a regular WordPress SEO tool! By using this plugin you will have your own WordPress SEO adviser right next to you 24/7. You just focus on writing and the plugin will do the SEO audit for each post and give you hints on what you should change in order to achieve best SEO results.

Out of the box solution for those who are not familiar with search engine optimization techniques, just install the plugin and you are ready to go! Intermediate and advanced users can make fine tuning to their websites (nofollow and noindex tags to specific pages and links, set up custom keywords and description to each post, revisit after tag, author tag, an many more).

== Installation ==

In order to install **Vibe SEO Pack** plugin you need to do the following:

1. Upload vibe-seo-pack folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to **Vibe SEO Pack** options page and add follow the instructions..
1. Good to GO!

== Frequently Asked Questions ==

= Google Analytics Tracker Is Not Working! Why? =

In this case your theme code is not written according to WordPress standards. To fix the problem go to your WordPress **Dashboard - Appearance - Editor** and select footer.php for editing. Now add this code: `<?php wp_footer(); ?>` bellow the `</body>` tag. Save changes and you're done. If your footer.php file contain encrypted code you should really think about changing your theme.
